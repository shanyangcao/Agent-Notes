# 是什么 / 原理 / 架构（LLM）

> 细化 LLM 的「是什么、基本原理、模型架构」三个维度的内容。

- ##  LLM是什么

  **LLM = Large Language Model（大语言模型）**

  简单理解：

  LLM 是一种可以 **理解和生成人类语言的人工智能模型**。

  比如你现在用的 **ChatGPT** 就是基于 LLM 构建的。

  它可以做很多事情，例如：

  - 回答问题
  - 写文章
  - 翻译语言
  - 写代码
  - 总结文档
  - 聊天对话

  常见的大语言模型有：

  - **GPT-4**
  - **LLaMA**
  - **Claude**
  - **Gemini**

  核心特点：

  - 训练数据非常大（互联网文本、书籍等）
  - 参数规模巨大（几十亿到几千亿参数）
  - 能理解上下文

  ## LLM原理

  LLM 的核心思想其实很简单：

  **根据上下文预测下一个最可能的词。**

  例如一句话：

  ```
  今天天气很好，我想去 ___
  ```

  模型会计算概率：

  - 公园（概率高）
  - 上班（概率中）
  - 月球（概率很低）

  最后选择概率最高的词。

  不断重复这个过程，就能生成完整文章。

  ### 核心技术

  LLM主要依赖一种模型架构：

  👉 **Transformer**

  Transformer 的关键机制是：

  **Attention（注意力机制）**

  意思是：

  模型在读一句话时，会自动判断哪些词更重要。

  比如：

  ```
  小明把书放在桌子上，因为它很重
  ```

  模型会判断：

  - “它”指的是 **书**
  - 而不是 **桌子**

  ## LLM架构

  一个完整的 LLM 系统一般包括几个部分：

  ### 数据（Training Data）

  来源：

  - 互联网网页
  - 书籍
  - 论文
  - 代码

  用于训练模型理解语言。

  ### 模型结构（Model）

  大部分 LLM 使用：

  👉 **Transformer**

  结构大致：

  ```
  输入文本
     ↓
  词向量 (Embedding)
     ↓
  Transformer层 (Attention)
     ↓
  输出概率
     ↓
  生成下一个词
  ```

  ### 训练过程（Training）

  训练步骤：

  1. 输入文本
  2. 隐藏部分词
  3. 让模型预测
  4. 计算误差
  5. 更新参数

  重复 **几万亿次训练**。

  ### 推理（Inference）

  训练完成后：

  用户输入：

  ```
  LLM是什么？
  ```

  模型输出：

  ```
  LLM是Large Language Model...
  ```

  这一步叫 **推理**。
