# 学习路线规划

> 结合自己的基础与目标，为「大模型 Agent / 后端基础」设计的一条学习路线。

## 阶段一：语言与基础

- Python 基础语法与标准库
- 简单脚本与小项目

## 阶段二：LLM 与 Agent 入门

- LLM 基础概念
- Prompt 基础
- 简单单轮对话与工具调用

## 阶段三：框架与工程化

- LangChain / Spring AI / FastAPI 等框架
- 项目结构、配置管理、日志与监控

## 阶段四：进阶与实践

- RAG、向量数据库
- MCP、Sub Agent、多 Agent 协作
- 结合自己的业务场景做一个完整 Demo

> 可以按时间轴或目标（例如「求职 / 项目落地 / 提升生产力」）继续细化路线。
