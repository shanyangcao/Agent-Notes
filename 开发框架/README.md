# 开发框架

> 记录使用各类框架（如 LangChain、FastAPI 等）时的经验与示例，方便搭建自己的 Agent / API 服务。

## LangChain

- 核心概念：链（Chain）、工具（Tool）、记忆（Memory）、Agent 等
- 常见用法示例
- 集成不同模型与向量数据库

## FastAPI

- 快速上手：路由、请求/响应模型
- 与 LLM / Agent 结合：封装成 HTTP API
- 中间件、依赖注入、配置管理

## 组合实践

- 使用 LangChain + FastAPI 搭建一个简单的对话或 RAG 服务
- 部署与调试时的注意事项
